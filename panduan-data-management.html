üéØ Cara Menggunakan Data Management

Step 1: Akses Data Management

1. Login sebagai Admin atau Superadmin

2. Klik menu "Data Management" di navigasi atas

3. Pilih salah satu opsi:

  • Import Data - Upload file Excel/CSV
  • Export Data - Download data dalam format CSV
  • Clear Data - Hapus semua data (dengan konfirmasi)
  • Data Statistics - Lihat ringkasan data

Step 2: Import Data dari Excel/CSV

1. Pilih "Import Data"

2. Pilih jenis data yang akan diimport:

  • üìã Pengajuan - Data pengajuan sertifikasi
  • üè¢ Kontraktor - Data perusahaan kontraktor
  • üë§ Personnel - Data pekerja/personnel
  • üèÜ Sertifikat - Data sertifikat yang diterbitkan

3. Upload file dengan cara:

  • Klik area upload untuk memilih file
  • Atau drag & drop file ke area upload

4. File yang didukung: .xlsx, .xls, .csv (maksimum 10MB)

Step 3: Mapping Kolom

1. Setelah file diupload, sistem akan menampilkan preview data

2. Lakukan mapping kolom dari file Excel/CSV ke field sistem

3. Kolom yang ditandai * adalah wajib

4. Sistem akan otomatis mencocokkan kolom berdasarkan nama yang mirip

5. Klik "Import Data" untuk memproses

Step 4: Data Tersimpan Otomatis

1. Data yang diimport akan langsung tersimpan di Local Storage browser

2. Data akan tetap ada meskipun browser ditutup

3. Dashboard akan otomatis update dengan data baru

4. Statistik akan menampilkan data real-time

üíæ Cara Kerja Local Storage

üîç Apa itu Local Storage?

Local Storage adalah fitur browser yang memungkinkan website menyimpan data di komputer Anda secara lokal. Data ini akan tetap tersimpan meskipun browser ditutup atau komputer di-restart.

‚úÖ Keuntungan Local Storage

  • Data tersimpan di laptop/komputer Anda secara lokal
  • Tidak memerlukan koneksi internet untuk mengakses data
  • Data tetap ada meskipun browser ditutup
  • Performa akses data sangat cepat
  • Tidak ada biaya hosting database

‚ö†Ô∏è Perhatian Penting

  • Data hanya tersimpan di browser dan komputer ini saja
  • Jika menghapus cache/data browser, data akan hilang
  • Tidak bisa diakses dari komputer/browser lain
  • Disarankan backup data secara berkala dengan Export

üìã Data Pengajuan

üîÑ Real-Time Data Updates

Bagaimana Data Update Real-Time?

1. Import Data: Setelah import berhasil, data langsung tersimpan di Local Storage

2. Auto Refresh: Dashboard dan statistik otomatis update

3. Persistent Storage: Data tetap ada saat buka aplikasi lagi

4. Merge Data: Data baru akan digabung dengan data existing

Melihat Perubahan Data

1. Kembali ke Dashboard setelah import

2. Lihat statistik yang sudah terupdate

3. Check tabel pengajuan terbaru

4. Grafik dan persentase otomatis terhitung ulang

üí° Tips dan Best Practices

‚úÖ Rekomendasi

  • Backup data secara berkala dengan fitur Export
  • Gunakan ID yang unik untuk setiap record
  • Pastikan format tanggal konsisten (YYYY-MM-DD)
  • Validasi data di Excel sebelum import
  • Import data sedikit demi sedikit untuk testing

‚ö†Ô∏è Hal yang Harus Dihindari

  • Menghapus cache browser tanpa backup
  • Menggunakan ID yang sama untuk record berbeda
  • Upload file yang terlalu besar (>10MB)
  • Format tanggal yang tidak konsisten
  • Kolom wajib yang kosong
